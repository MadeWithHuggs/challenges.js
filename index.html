Challenge 1

FizzBuzz

Write a short program that inserts each number from 1 to 100 into an array. For each multiple of 3, print "Fizz" instead of the number. For each multiple of 5, print "Buzz" instead of the number. For numbers which are multiples of both 3 and 5, insert "FizzBuzz" instead of the number.


var output = [];
var count = 1

function fizzBuzz() {

    while(count <= 100){
        if (count % 3 === 0 && count % 5 === 0) {
        output.push("FizzBuzz");
    } else if (count % 3 === 0){
        output.push("Fizz");
    } else if (count % 5 === 0) {
        output.push("Buzz")
    } else {
        output.push(count);
    } 
    
    count++;
            
    console.log(output);
    }
}

BMI Calculator

Write a JavaScript function calculateBMI that:

Takes two arguments: weight (in kilograms) and height (in meters).

Calculates the Body Mass Index (BMI) using the formula:

BMI = weight / height2

Returns a string indicating the BMI category based on the calculated BMI value:

Underweight: BMI < 18.5
Normal weight: 18.5 ≤ BMI < 24.9
Overweight: 25 ≤ BMI < 29.9
Obesity: BMI ≥ 30

Expected Output If you call calculateBMI(70, 1.75), it should return "Normal weight".


function bmiCalculator(weight, height) {
var bmi = Math.floor(weight / (height * height)) 

if(bmi < 18.5){
        return("Your BMI is " + bmi + ", so you are underweight.")
    } else if(bmi > 18.5 && bmi < 24.9) {
        return("Your BMI is " + bmi + ", so you are normal weight.")
    } else if (bmi > 24.9){
        return("Your BMI is " + bmi + ", so you are overweight.")
    } 
}

Challenge 7

Leap Year Calculator

Write a JavaScript function isLeap() that:

1. Takes a single argument, year, representing the year to check.

2. Return "Leap year." if the year is a leap year, and "Not leap year." if it is now.

3. Follows the rules of identifying a leap year:

  • A year is a leap year if it is divisible by 4
  • However, if the year is also divisible by 100, it is not a leap year unless it is also divisible by 400.

Expected output, if you call isLeap(2024), it should return "Leap year." If you call isLeapYear(1900), it should return "Not leap year", since 1900 is divisible by 100 but not by 400. 


function isLeap(year) {

    if (year % 4 == 0 || year % 100 != 0 && year % 400 == 0) {
            return ("Leap year.")
    } else {
        return ("Not leap year.")
    }
    
}

Who's Buying Lunch?

Create an array with a list of names to simulate a group that you are at a restaurant with. Use a JaveScript function to select a random name from the array and return it. 


function whosPaying(names) {

    var names = ["Angela", "Ben", "Jenny", "Michael", "Chloe"]
    var listLength = names.length;
    var randomNum = Math.floor(Math.random() * listLength);
    var payee = names[randomNum];
    
    return(payee);

    
}
